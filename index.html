📖 API 與授權設定教學

🔑 YouTube Data API 金鑰 (抓影片留言、轉換頻道 ID 用)

  1. 前往 Google Cloud Console 並登入您的 Google 帳號。
  2. 點擊左上角「選取專案」，然後點擊「新增專案」建立。
  3. 在上方搜尋欄搜尋「YouTube Data API v3」並點擊進入後按「啟用」。
  4. 在左側導覽列點選「API 和服務」 > 「憑證」。
  5. 點擊上方的「+ 建立憑證」，並選擇「API 金鑰」。
  6. 取得並複製該串金鑰，貼回家本程式畫面中的**「YouTube Data API 金鑰」**欄位即可！

🔐 OAuth Client ID (抓取專屬付費會員清單用)

  1. 同樣在 Google Cloud Console，點擊左側「API 和服務」 > 「OAuth 同意畫面」。
  2. 選擇「外部」並點選建立，填寫必填的「應用程式名稱」、「使用者支援電子郵件」及最下方的開發人員電子郵件，其餘留空即可，一路按「儲存並繼續」到底。
  3. 回到左側導覽列點選「API 和服務」 > 「憑證」。
  4. 點擊上方的「+ 建立憑證」，選擇「OAuth 用戶端 ID」。
  5. 「應用程式類型」選擇為「網頁應用程式」。
  6. 在「已授權的 JavaScript 來源」與「已授權的重新導向 URI」，輸入您目前運行伺服器的網址（必填：例如 http://localhost:8081）。
  7. 點擊建立後，複製這串 用戶端 ID (通常以 .apps.googleusercontent.com 結尾)，貼回家本程式的對應欄位！
補充提醒： 由於嚴苛的安全性限制，OAuth 授權流程無法在直接雙擊右鍵執行的 file:// 本地檔案環境下運作。若您要使用自動抓取會員功能，必須將網頁放置於您的伺服器，或利用簡單的指令（如 python -m http.server）開啟網站。
